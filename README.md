# Cholera Contagion Source Code

Cholera Contagion is a game I made for the Extra Credit game jam. The game is based on the 1854 cholera outbreak in London, and is a mix of Where In
World Is Carmon Sandiago and Minesweeper. The game was made using Phaser 3 project template with ES6 support via [Babel 7](https://babeljs.io/) and [Webpack 4](https://webpack.js.org/)


## Requirements

[Node.js](https://nodejs.org) is required to install dependencies and run scripts via `npm`.

## Available Commands

| Command | Description |
|---------|-------------|
| `npm install` | Install project dependencies |
| `npm start` | Build project and open web server running project |
| `npm run build` | Builds code bundle with production settings (minification, uglification, etc..) |
